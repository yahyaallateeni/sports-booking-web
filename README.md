# Sports Booking Web

واجهة ويب أولية لمنصة حجز ملاعب رياضية مبنية باستخدام Next.js + TypeScript + Tailwind CSS.

## التشغيل

```bash
npm install
npm run dev
```

ثم افتح:

```bash
http://localhost:3000
```

## الصفحات الحالية

- `/` الرئيسية
- `/venues` قائمة الملاعب
- `/venues/1` صفحة تفاصيل ملعب
- `/owner` لوحة المالك
- `/admin` لوحة الأدمن
- `/api/health` فحص بسيط للواجهة

## الخطوة القادمة

- صفحات تسجيل الدخول والتسجيل
- صفحة الحجز والدفع
- فصل layout خاص بالداشبورد
- ربط البيانات لاحقًا مع backend API
